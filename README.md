# MY BENCH APP
 
ベンチプレスの記録ができる自己満アプリ(SPA)です。
 
## Features
 
Vue.js
 
## Requirement
 
* node v18.15.0
 
## Installation
 
```bash
git clone https://github.com/yu03-dev/my-bench-app.git
cd {my-bench-app}
npm install
echo VITE_ENV=development >> .env
```
 
## Usage
 
```bash
npm run build
npm run preview
```

"http://localhost:4173"にアクセス
 
## Note
 
このアプリでできることは以下の通りです。

* '/': ホームページ
* '/signin': サインイン画面
* '/signup': サインアップ画面
* '/mypage': マイページ画面(ベンチプレスの記録の一覧、作成、編集、削除)
